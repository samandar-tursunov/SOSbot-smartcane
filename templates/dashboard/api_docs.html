📡 API Hujjatlari

Tashqi Telegram bot location qabul qilganda sizning serveringizga push qiladi

🔄 Oqim

  1. Foydalanuvchi tashqi Telegram botga 📍 location yuboradi
  2. Tashqi bot shu koordinatalarni POST qiladi → bu endpoint
  3. Server eng yaqin tumanni topadi
  4. Bizning bot tuman Telegram guruhiga xabar yuboradi

🔐 Autentifikatsiya

Har bir so'rovda X-API-Key header yuborilishi shart:

X-API-Key: your-secure-api-key-for-external-service

Alternativ: Authorization: Bearer <SOS_API_KEY>

🆘 SOS Location — POST

POST {{ endpoint }}

Minimal request

{
  "latitude": 41.3110810,
  "longitude": 69.2405620
}

Tavsiya etilgan (Telegram ma'lumotlari bilan)

{
  "location": {
    "latitude": 41.3110810,
    "longitude": 69.2405620
  },
  "telegram_user_id": 123456789,
  "telegram_username": "foydalanuvchi",
  "telegram_first_name": "Ali",
  "telegram_message_id": 42,
  "telegram_chat_id": 123456789
}
Maydon Turi Majburiy Tavsif
latitude decimal ✅ SOS kenglik (-90 … 90)
longitude decimal ✅ SOS uzunlik (-180 … 180)
location object ✅* Telegram location obyekti
telegram_user_id int ❌ message.from.id
telegram_username string ❌ message.from.username
telegram_message_id int ❌ message.message_id
telegram_chat_id int ❌ message.chat.id
source_id string ❌ Avtomatik: tg-{chat_id}-{message_id}
notes string ❌ Qo'shimcha izoh

Xato javoblar

# 401 — API kalit noto'g'ri
{"success": false, "error": "Unauthorized — invalid or missing API key"}

# 400 — validatsiya xatosi
{"success": false, "errors": {"latitude": ["..."]}}

# 422 — tuman/guruh topilmadi yoki Telegram xatosi
{"success": false, "alert_id": 43, "status": "no_group", "error": "..."}

⚙️ Ishlash tartibi

  1. Tashqi servis SOS latitude va longitude yuboradi
  2. Server nuqtani 10 ta tuman chegaralari bilan solishtiradi
  3. Chegara ichida bo'lsa — o'sha tuman; aks holda eng yaqin markaz tanlanadi
  4. Tumanga biriktirilgan Telegram guruhiga matn + location yuboriladi
  5. Natija bazaga saqlanadi va API javob qaytariladi
